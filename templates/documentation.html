Getting started

Authentication
Using this API requires a user account and a temporary access token. Sign in with Google to create or look up your user account and get a temporary access token. Be sure to include your temporary access token in the Authorization header of your request.
Media Types
This API accepts and returns only JSON in the request and response bodies. When making a request with contents in the response body, we strongly encourage you to explicitly include "application/json" in the Accept header of your request. When sending a request with contents in the request body, include "application/json" in the Content-Type header of your request as well.
Contributors
By default, users are not set as contributors. A contributor is someone who can add, edit, and delete resources. This documentation covers some methods that are exclusive to users with contributor capabilities. These methods are provided in the public documentation for the sole purpose of demonstrating the full functionality of this REST API.
